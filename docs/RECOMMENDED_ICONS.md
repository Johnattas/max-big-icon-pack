## Como efetivar

- **atom / AFileIcon**: já são fontes do pipeline (`scripts/import-icons/`). Estes foram antes ignorados por já terem associação de extensão no Material; para adicioná-los como ícone próprio, criar override associando extensão/nome ao novo `mbip-<slug>`.
- **simple-icons / devicon**: novo estágio no pipeline lendo o SVG do pacote npm — para simple-icons (monocromático) aplicar o `hex` de marca oficial que o pacote expõe; gera `mbip-<slug>.svg` e mapeia extensão/nome/pasta.

Posso implementar o estágio simple-icons/devicon no pipeline para efetivar estes 400, se quiser.
