## 💡 O que é

O **Max Big Icon Pack** parte da elegância flat do consagrado **Material Icon Theme** e o expande com uma curadoria de **mais de 10.000 ícones** — todos no mesmo estilo visual coerente. O objetivo é simples: **qualquer arquivo ou pasta que você abrir deve ter um ícone que faça sentido.** Do `.pinia` ao `dockerfile`, da pasta `stripe/` à `worktrees/`, o reconhecimento visual é imediato.

## ▶️ Como ativar

1. Abra a Paleta de Comandos (`Ctrl+Shift+P` / `Cmd+Shift+P`).
2. Digite **`Preferences: File Icon Theme`**.
3. Selecione **`Max Big Icon Pack`**.

## ✨ Destaques

- 💎 **Biblioteca gigante (10.240+ ícones)** — cobre praticamente qualquer linguagem, framework, banco, ferramenta de DevOps, serviço ou marca.
- 🎨 **Estilo 100% coerente** — tudo flat, no espírito do Material. Marcas viram ícones de pasta com o logo **na cor da marca + badge branco** (fechada e aberta).
- 🧠 **Reconhecimento inteligente de pastas** — `storage`, `http`, `ai`, `traits`, `observers`, `workflows`, `webhooks`, `sessions`, `requests`, `leads`, `.superpowers`, `mcp` e dezenas de outras já mapeadas.
- 🚦 **Manifestos vs. locks coloridos** — `package.json` 🟢 e `package-lock.json` 🔴; idem `composer.json`/`composer.lock`.

## 📚 De onde vêm os ícones

Curadoria de bibliotecas open-source flat, gerada por um **pipeline reproduzível**: **Material Icon Theme** (base), **file-icons/Atom**, **A File Icon**, **Simple Icons** (~3.000 marcas) e **Devicon**.
